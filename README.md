# Vrinda Store Data Analysis

## 📊 Overview
This repository contains an Excel-based analysis of Vrinda Store's sales and customer data.  
The project focuses on identifying patterns, trends, and insights to support better business decisions.

## 🛠️ Tools Used
- Microsoft Excel (Pivot Tables, Charts, Data Cleaning)
- Power Query (optional, if used)
- Basic statistical analysis

## 📈 Key Insights
- Monthly sales trends and seasonality
- City-wise and gender-wise customer distribution
- Top-performing product categories
- Recommendations for improving sales strategy

## 🚀 How to Use
- Download the Excel file:  
   👉 [Vrinda Store Data Analysis.xlsx](https://github.com/aditya-g18/Vrinda-Store-Intractive-Dashboard-Using-Excel/blob/main/Vrinda%20Store%20Data%20Analysis.xlsx)
- Open in Microsoft Excel.
- Navigate through different sheets and dashboards for insights.
